# 🎮 Gamified TODO App

自己管理型RPG：計画への報酬とリソース管理によるバーンアウト防止を実現したTODOアプリケーション。

## 📖 概要

従来のTODO管理に加え、ユーザーの「計画性」「実行の質」「自己ケア」をHP/MPなどのRPGステータスとして可視化し、ゲームのインセンティブによって生産性と健全な習慣化を同時に促進するユーティリティアプリです。

### 🎯 主な特徴

- **未確定報酬システム**: タスクに詳細を設定するたびに報酬がストックされ、完了時に確定
- **HP/MPリソース管理**: 体力と精神力を可視化し、バーンアウトを防止
- **プロジェクト階層構造**: タスクを柔軟にプロジェクト化・サブタスク化
- **ポモドーロタイマー**: 集中作業とMP管理を連携
- **タグシステム**: 回復タスク、メンタルケア、学習タスクなどを分類

## 🎲 ゲームシステム

### ステータス

| ステータス | 説明 |
|---|---|
| **HP (体力)** | 実行力。0になると新規タスク登録がブロック。回復タスクで回復。 |
| **MP (精神力)** | 集中力・意欲。報酬確定率に影響。メンタルケアタスクで回復。 |
| **INT (知力)** | 計画の質。未確定報酬のストック上限に影響。プロジェクト分解で向上。 |
| **Speed** | 実行効率。短期タスクの報酬に影響。素早い完了で向上。 |

### 報酬システム

1. **未確定報酬**: タスク設定時に詳細に応じてポイントがストック
   - サイズ設定: 10pt (Small) / 25pt (Medium) / 50pt (Large)
   - 期限設定: +5pt
   - 優先度設定: +3pt × 優先度
   - タグ設定: +2pt × タグ数

2. **確定報酬**: タスク完了時にストックされた報酬を獲得
   - MPによる倍率ボーナス
   - スピードボーナス（24時間以内完了で1.3倍）
   - 優先度ボーナス

3. **分解ボーナス**: Largeタスクを3個以上のサブタスクに分解で追加XP

## 🚀 セットアップ

### 必要要件

- Node.js 18.x以上
- Expo CLI
- iOS/Androidシミュレーターまたは実機

### インストール

```bash
# 依存関係のインストール
npm install

# 開発サーバーの起動
npm start

# iOSで実行
npm run ios

# Androidで実行
npm run android
```

## 📱 画面構成

### 1. ホーム画面 (GamifiedHomeScreen)
- タスク一覧表示
- ステータスバー（HP/MP/XP）
- フィルター機能（全て/未完了/完了）
- プロジェクト昇格機能

### 2. タスク編集画面 (GamifiedTaskFormScreen)
- タスク名・詳細入力
- サイズ選択（S/M/L）

- 期限設定
- タグ設定（回復/メンタルケア/学習）
- 報酬プレビュー

### 3. ステータス画面 (StatusScreen)
- HP/MPバー表示
- 累積経験値（XP）
- INT/Speedレベル
- ステータス向上のヒント

### 4. ポモドーロ画面 (PomodoroScreen)
- 25分タイマー
- 完了でMP +10
- 中断でMP -5ペナルティ
- 休憩タイマー

## 🎮 使い方

### タスクの作成
1. 右下の「+」ボタンをタップ
2. タスク名、サイズ、優先度を設定
3. 必要に応じてタグを設定
4. 報酬プレビューで獲得ポイントを確認
5. 保存

### タスクの完了
1. タスクのチェックボックスをタップ
2. 未確定報酬が確定し、XPを獲得
3. タグに応じてHP/MPが回復

### プロジェクト化
1. タスクの「📁 プロジェクトに昇格」をタップ
2. サブタスクを追加（分解ボーナス獲得）
3. 全サブタスク完了でプロジェクト完了

### ポモドーロの使用
1. 🍅ボタンをタップ
2. 25分間集中作業
3. 完了でMP +10獲得
4. 休憩を取る
